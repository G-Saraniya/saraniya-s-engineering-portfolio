# Saraniya G - Engineering Portfolio

A modern, interactive portfolio website for Saraniya G, an Electronics & Communication Engineering graduate specializing in IoT, Embedded Systems, and Software Engineering.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Features

- **Interactive UI**: Hover effects, smooth transitions, and dynamic content revealing.
- **Smart Chatbot**: "Ask Me Anything" feature with auto-suggestions and dynamic responses based on portfolio data.
- **Project Showcase**: Detailed project cards with expand/collapse functionality.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dark/Light Mode**: Theme toggle for user preference.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/G-Saraniya/saraniya-s-engineering-portfolio.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
